Origin 直播公开课

发表于 2024-03-07 更新于 2024-06-07 分类于 Science

前言

Origin 是一款广受科研工作者和数据分析师欢迎的专业绘图和数据分析软件。无论您是刚接触Origin，还是希望提高您的数据处理和图形制作技能，其直播课程都将为您提供全面、深入的指导。

2020.03.06

主题

  • OriginPro 助力科研论文

2020.03.20

主题

  • Origin 入门

内容

本次直播会分享Origin 的学习资源， 并以全球疫情数据为例子，展示
https://www.bilibili.com/read/cv5186918

  1. 如何导入数据并保持连接可随时更新
  2. 如何制作符合期刊出版要求的图形
  3. 如何绘制动图

2020.05.29

主题

  • 拼图专题

内容

本次直播会对各种拼图方法的优劣进行比较和介绍，以及手把手展示如何在 Origin 中进行拼图。如何在拼图后对图层进行重组和修改 。内容包括：

  1. 规则的 N×M 拼图
  2. 不规则的图片组合
  3. 含三维图的拼图
  4. 拼图后对图层的重组和修改

2020.06.12

主题

  • 拟合专题

内容

深入探讨怎么样在 Origin 中做拟合

  1. 如何选择拟合工具
  2. 如何根据你的数据选择拟合函数
  3. 怎样更好地拟合？ 探讨怎样设置初始值，怎样判断拟合得好不好，为什么拟合不成功等问题
  4. 怎么样设置出想要的拟合结果

互动问答

  • 主要回答了关于隐函数拟合，回归，分峰拟合等用户问题

2020.07.03

主题

  • XRD 科研绘图

内容

  • XRD 科研绘图专题：主要介绍在科研绘图中，不同的数据适合用什么类型的图来表达， 并深入以实例介绍 XRD 数据的不同的绘图表达方式以及美观美化。

2020.07.17

主题

  • Origin 和 Python 的交互使用

内容

  • 这个视频介绍了如何使用 Origin 内置 Python 进行数据处理和分析。

2020.07.31

主题

  • 波谱分析 (Peak Analysis)

内容

  • 本次直播介绍了对于光谱曲线，如何处理基线 (XPS, ALS)，寻峰(多峰，隐峰)，分峰积分和拟合，以及对于多光谱的批量峰值分析；
  • 批量扣除基线（第 17 分钟）。

2020.07.31

主题

  • 分峰绘图 (Peak Analysis)

内容

  • 这个视频详细介绍了常见的分峰绘图的设置和画法，包括了曲线图式， 画出基线的面积图式，没有基线的面积图式，三维层叠式。

2020.08.14

主题

  • Python语法基础介绍

内容

  • 这个视频是 2020 年 8 月 14 OriginPro 直播公开课的第一部分， 主要介绍了 Python 最基本的语法基础。

2020.08.14

主题

  • Origin 与 Python 交互

内容

  • 这个视频是 2020 年 8 月 14 OriginPro 直播公开课的第二部分， 对 OriginPro 和 Python 的交互使用的基础用法做了详细介绍。并用实例展示了如何使用 Python 画图， 用 Python 函数设置列值， 做拟合函数等。

2020.08.28

主题

  • 多图层绘制攻略

内容

  • 掌握多图层绘制和设置方法：包括多轴图以及组合图的原理，绘制，调整和设置。
