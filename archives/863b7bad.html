Origin 绘制 3D 图时无法正常显示图例

发表于 2023-04-25 更新于 2024-01-20 分类于 Science

现象

Origin 部分版本在绘制图形时，图例等部分会出现如下图所示的异常的黑色色块，此问题与 AMD 显卡 OpenGL 与 Origin 的兼容性有关，本文将简述解决方案。

官方解决方案

Origin 在 2023b 中修复了这个问题：ORG-24094

  1. 将软件更新至 Origin 2023b 及以上版本；
  2. 通过 Preference -> System Variables 打开系统变量设置窗口；
  3. 设置系统变量OGLDP的值为1；
  4. 重新打开软件。

如果上述方法无法解决问题，可尝试通过 Preference -> 3D OpenGL Settings 取消勾选 Enable Display List。

参考文档：Strange-Color-Block-On-3D-Graph

坊间解决方案

以下方法为可能有效的方案，不保证解决问题。

  • 如果你是双显卡用户，可以将 Origin 调用的显卡切换为另一张显卡；
  • 尝试将 AMD 显卡的驱动程序更新到最新，或降级到上一个版本。

环境

  1. Windows 11 (22621);
  2. Origin 2023b beta2 (Academic).
