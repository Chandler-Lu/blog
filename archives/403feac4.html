Jade 6.5 图文安装教程

发表于 2017-11-23 更新于 2023-08-23 分类于 Science

前言

Jade 是一种功能强大、全功能的粉末 XRD 图处理和分析软件，尤其注重定量分析和鉴定物相，可实现非常规少量物相的准确鉴定和定量分析。考虑到国内购买 Jade 的机构数量极其有限，本文将介绍 Jade 6.5 的安装方法，但不会提供安装程序。

名词解释

  1. 你下载下来并解压的文件，以下称为”安装包”。
  2. 安装时让你选择的路径，以下称为”安装目录”。

账户名查看

打开任务管理器，找到”用户”，这就是你的账户名（务必注意大小写和特殊符号），本步骤下面会用到。

准备工作

  1. 关闭所有安全软件以防在修改注册表时被拒绝。
  2. 确保目前登陆账户不是 Administrator（上面说了怎么查看账户名）。
  3. 将安装文件拷贝至不含任何中文及特殊符号的目录中。
  4. 下载后务必将安装包解压！

安装

  1. 选择安装路径，点击安装，直至安装完成。安装结束后，窗口会自动关闭。

控件注册

安装结束后，软件是无法正常运行的，此时我们要注册对象链接和嵌入用户控件（OCX）。

  1. 在安装包中找到”ocx”文件夹，编辑”注册 ocx.bat”；

  2. 将第一行修改为安装盘符（安装路径前 2 个字符，如 C:），第二行修改为”cd 安装路径”，如图所示，然后保存；

  3. 管理员模式运行”注册 ocx.bat”，接着会提示多个”xxxx 已成功”，全部点击确定直到窗口关闭；

备注

在进行以上步骤时，如果 grid32.ocx 和 MSCOMCTL.OCX 安装报错，请将安装包中所有的 .ocx 文件拷贝至 C:\Windows\SysWOW64（假设系统盘是 C 盘）后，重做第二、三步。此时，将第一行修改为安装盘符 C:，第二行修改为 cd C:\Windows\SysWOW64，然后保存，接着以管理员模式运行。

添加注册表

本步骤务必注意大小写。

  1. 使用”win+R”组合键打开”运行”，键入以下命令，打开注册表编辑器；

  1. 展开”HKEY_LOCAL_MACHINE”，展开”SOFTWARE”，右键”SOFTWARE”，选择”添加项”，命名为”MDI”；

  2. 在”MDI”下新建一个新项，名为”license”；

  3. 在”license”左侧新建一个”字符串值”，命名为”Owner”；

  4. 修改”Owner”的值为你的用户名。

配置 PDF 卡片

  1. 随便打开一个示例文件（打开你自己的也行），点击上方”pdf”，选择”setup”；

  2. 点击如图所示按钮，在安装包中找到”pdf2.dat”文件，这个文件在安装包的”jade65/PDF22004”中；

  3. 选定后，点击”select all”，然后”create”；

  4. 由于本步骤时间很长，开始时会提示是否在完成后关机/注销，选择否；

  5. 全部完成后，将提示（已知部分系统在载入 pdf 卡片时出现 Run-time error ‘75’的报错，原因是没有管理员权限，无法写入相关数据。解决方法是，以管理员模式启动 Jade（以后不再需要，此问题多发于 win7 以上系统）；

  6. 如果可以正常查找，则安装成功。

环境

虚拟机：Parallels Desktop 13
操作系统环境：基于 MSDN 的 win10 专业版（1703）
