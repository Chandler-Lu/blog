如何在 macOS 虚拟机上安装 OriginLab

发表于 2023-08-07 更新于 2023-08-08 分类于 Science

前言

随着 Apple silicon 平台的普及，越来越多的用户开始使用 macOS 系统。然而，Origin 仅在 Windows 平台上进行了适配，对于 macOS 平台的用户则需要通过虚拟机进行安装。本文将从 Windows 的下载开始，详解 Origin 在 macOS 平台上的安装步骤。

在 macOS 平台部署虚拟机并安装 Origin 具有一定的成本，如果您只需要浏览数据和图片，不妨尝试由 OriginLab 开发的免费软件 Origin Viewer，这可以使您在 macOS 上原生查看和复制 Origin 项目文件中包含的信息。

说明

  1. 阅读此教程前，建议你具备基本的问题处理能力和全球互联网的访问能力；
  2. 由于安全问题，所有软件均只提供官网或可信渠道的下载方式，本文不保证任何来自于第三方（含盗版）渠道的软件可以正常使用。互联网平台上存在大量第三方修改版的操作系统，用户无法知道这样的系统中加了什么料，如果底层环境都不可信，在其上层运行的软件的稳定性和安全性更是无从谈起；
  3. 当前能在 Apple silicon 平台上运行 Origin 的虚拟机只有 Parallels Desktop 和 VMware Fusion 13，本文将以 Parallels Desktop 为例讲解。另外，Crossover 不具备完整的运行环境，无法适配 OriginLab；
  4. 由于架构的特殊性，Origin 在 Apple silicon 上运行存在 OriginLab、虚拟机、Windows、macOS 四方的兼容问题，本文仅保证撰文时所使用的软件和系统版本可以完美运行；
  5. 虚拟机和 Windows 本体需要占用至少 30 GB 的空间，Windows 的运行内存建议大于 4 GB，故不建议在配置过低的设备上尝试；
  6. 本文对于 Intel 平台的 macOS 也适用，Intel 平台直接使用 x86-64 架构的 Windows 10/11 即可。

架构

  1. x86, ARM：处理器指令架构，生活中常见的 Intel 和 AMD 的 CPU 大多为 x86 架构，而 Apple silicon 为 ARM 架构；
  2. x86-64：基于 x86 架构的 64 bit 扩展。注：Origin 在 2020 年之后的版本中取消了对 32 bit 的支持，故新版本只能运行在兼容 x86-64 架构的平台上；
  3. Windows 11 on ARM 能够运行 x86-64 的应用程序，这也是 Origin 64 bit 能够在 ARM 架构上运行的基础。

Window 11 for ARM 特别说明

  1. 微软官方已经使用 Win 11 for ARM 替换了 Win 10 for ARM，也就是不再对 Win 10 for ARM 提供支持，与此同时，Win 10 for ARM 仅在测试版中对 64 bit 的 x86 应用程序进行了兼容，故不推荐安装 Win 10。当然，如果您使用的是早期 Intel CPU 的 macOS，则可以根据喜好选择 X86 64 bit 的 Win 10 或 Win 11；
  2. 原则上，本文仅推荐通过官方渠道获取软件和操作系统，但微软仅公开了 Win 11 for ARM Insider Preview（测试版）的下载方式，而其正式版并不提供给终端用户，因此才会有下文的第三方下载渠道。

虚拟机的安装

  • 前往 Paralles Desktop（下简称“PD”）官网下载安装程序，启动后直至下载安装完成，并激活 PD。

下载和安装 Windows 11

PD 自带方案

PD 自带了 Win 11 for ARM64 的下载途径，可以迅速完成操作系统的安装，当然，如果你对 Win 有更高的定制要求，可以阅读下方通过 UUP dump 下载的方案。

UUP dump 方案

UUP dump（下简称“UUP”）是一个足够可信的系统下载平台，所下载的系统来自微软官方。UUP 本可以在全平台进行下载，但笔者实际测试发现 UUP 在 macOS 上下载及其繁琐，故建议在其他的 Windows 上完成操作后，再将系统转移至需要安装虚拟机的机器中。

下载

本小节在 Windows 环境中完成。

  1. 进入 UUP 官网，选择正式发行的 arm64 架构版本；
  1. 选择当前最新的 Win 11 for ARM；
  1. 选择操作系统语言；
  1. 选择所需的操作系统版本；
  1. 启用 ISO 转换，并下载；
  1. 下载解压后，启动用于 Windows 平台的下载器，软件将自动进行下载前准备，系统组件的下载，和 ISO 镜像的转换工作；
  1. 下载和转换结束后，将获得用于虚拟机中安装的操作系统镜像。

安装

  1. 选择从 DVD 或镜像文件安装 Windows；
  1. 选择已经准备好的镜像文件（软件会自动搜索，如果未搜索到，可以手动选择）；
  1. 选择安装位置，如果你有个性化的需求，可以在安装前手动设定 CPU 核数和内存等；
  1. 等待 Windows 安装完毕；
  1. 安装完毕后，就可以看到熟悉的 Windows 系统了。

Origin 的安装

  1. 将 Origin 安装文件拖动至 PD 内。PD 在默认设置下，允许用户直接从 Finder 拖动文件；
  1. 将安装文件解压缩；
  1. OriginLab 官方已经提供了足够详细的安装文档和问题解决方案，这里不再对安装流程进行赘述；
  2. 按本文教程安装后的 Origin 可以直接使用，如果遇到闪退问题，请参考https://www.originlab.com/doc/en/Quick-Help/Crash-after-Windows-Update

环境

  1. macOS 13.5 on Mac mini 2023 16G；
  2. Paralles Desktop: 18.3.2 (Standard Edition)；
  3. Window 11 for ARM: 22H2 22621 (UUP dump 方案)；
  4. OriginPro 2023b (Academic)。
